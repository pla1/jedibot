Commands: hpr, nasa, xkcd, date, ping 8.8.8.8, help

hpr - Get the latest Hacker Public Radio episode

nasa - Get NASA's latest Image of the Day

xkcd - Get latest XKCD comic

date - Get the date in the Eastern time zone or supply a timezone like: date Europe/Paris
    Timezone list: https://paste.ubuntu.com/p/thZHPZkdrd/

ping - PING an ip address or host name
